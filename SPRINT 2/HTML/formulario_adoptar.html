Requisitos básicos para adopción

1. Ser mayor de edad. 2. Fotocopia cédula de ciudadanía. 3. Disponer de 45- 60 minutos de tiempo para realizar el proceso de adopción 4. Disponer de tiempo, espacio y recursos suficientes. 5. Contar con la aprobación de la Familia. 6. Aprobar el proceso de adopción (entrevista). 7. Contar con espacio suficiente para convivir con el animal. 8. Llevar correa para perro o guacal para gato.
